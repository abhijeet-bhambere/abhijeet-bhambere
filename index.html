1.3 Attributes: Intro & general practices

-HTML attributes provide additional information about HTML elements
-They're always specified in the start tag of an element
-Usually come in name/value pairs like: name="value"

(a) lang: Always incorporate the "lang" attribute with its country code inside the <"html"> tag to declare the language of the Web page.
This is meant to assist search engines and browsers.

(b)style: This attribute is used to format the element with respect to its size/color/font. It can be used to set the theme of the web page.

(b)title: The value of the title attribute will be displayed as a tooltip when you hover mouse over the element.
Check it out now! hover the mouse on the title to see this attribute work

1.4 Quotations & Citations:

Blockquote: A section quoted from another source; use the cite attribute to specify the source.
But additional formatting/highlights needed to further make the Blockquote stand out. Blockquote is usually indented by browsers

example:

example:WWF's goal is to: Build a future where people live in harmony with nature.
